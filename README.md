# domino-engine-js
A classic domino game engine written in pure JavaScript with an AI opponent and dynamic board layout.

This project was originally developed as a personal experiment to simulate a real domino table in the browser. The engine renders domino tiles dynamically using CSS and implements a simple AI capable of choosing playable tiles.

The goal of the project is to demonstrate how a complete board game can be implemented using plain JavaScript without frameworks.

Features
* Domino game engine written in pure JavaScript
* Simple AI opponent
* Dynamic domino table layout
* Domino tiles rendered using CSS & images
* Lightweight and framework-free
* Runs entirely in the browser

Technologies Used
* JavaScript
* HTML
* CSS
* jQuery


How to Run
 * Clone the repository
 * git clone https://github.com/YOUR_USERNAME/domino-engine-js.git

Open the project folder
Run the game by opening:

index.html

in your browser.

No build tools or installation steps are required.
